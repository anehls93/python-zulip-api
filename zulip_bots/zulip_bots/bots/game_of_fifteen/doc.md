# game_of_fifteen bot

Game of Fifteen bot is a bot designed to let you play a game of fifteen. To summon it, simply type `@game_of_fifteen`,
like so:

```
@game_of_fifteen
```

Run this bot as described in [here](https://zulipchat.com/api/running-bots#running-a-bot).
